# 🔒 VLESS VPN Controller - Инструкция

## 🚀 Быстрый старт

### 1. Запустить меню выбора локации:
```bash
vpn-ctrl
# или
python3 /home/kostik/vpn-client-aggregator/vpn-controller.py
```

### 2. Выбрать локацию из списка:
```
1. 🇩🇪 Germany (150 серверов)
2. 🇳🇱 Netherlands (120 серверов)
3. 🇺🇸 USA (80 серверов)
...
```

### 3. Готово! VPN работает в фоне

## ⚙️ Автозапуск

VPN работает **автоматически в фоне** после выбора локации.

### При загрузке системы:

VPN уже настроен на автозапуск через `~/.config/autostart/`

### После перезагрузки:

1. VPN запустится автоматически
2. Подключится к последней выбранной локации
3. Если нужно сменить - используйте `vpn-ctrl`


---

## 🔧 Как это работает

1. **Первый запуск:**
   - Запускаете `vpn-ctrl`
   - Выбираете локацию
   - VPN подключается и работает в фоне

2. **После выбора:**
   - VPN работает автоматически
   - Переподключается при обрыве
   - Обновляет серверы каждые 5 минут

3. **Смена локации:**
   - Запускаете `vpn-ctrl`
   - Выбираете новую локацию
   - VPN переподключается

## 🐛 Решение проблем

### VPN не подключается:
```bash
# Обновить список серверов
vpn-ctrl
# 4 (Обновить список серверов)

# Выбрать другую локацию
vpn-ctrl
# 1 (Выбрать локацию)
```

### Проверить логи:
```bash
tail -30 ~/vpn-client/logs/client.log
```

### Остановить VPN:
```bash
pkill -f "vless-vpn"
pkill -f "xray"
```
