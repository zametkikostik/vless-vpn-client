# 🚀 Быстрый поиск серверов

## ✅ Новая функция!

Теперь можно быстро найти серверы с низким пингом!

---

## 📝 Использование

### Быстрое сканирование:

```bash
vless-vpn-ultimate fast-scan
```

**Что делает:**
- ✅ Проверяет пинг всех серверов
- ✅ Сортирует по скорости
- ✅ Сохраняет топ-50 быстрых серверов
- ✅ Показывает статистику

- Статистика

---

## 🔌 Как использовать быстрые серверы

### Вариант 1: Автоматически

После сканирования просто запустите VPN:
```bash
vless-vpn-ultimate start
```

VPN автоматически выберет быстрый сервер!

### Вариант 2: Через GUI

```bash
vless-vpn-gui gui
```

В GUI выберите сервер с минимальным пингом.

---

## ⚡ Быстрые команды

```bash
# Быстрое сканирование
vless-vpn-ultimate fast-scan

# Обновить серверы и сканировать
vless-vpn-ultimate update
vless-vpn-ultimate fast-scan

# Запустить VPN с быстрым сервером
vless-vpn-ultimate start
```
