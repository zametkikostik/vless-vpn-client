# 🔥 СРОЧНО: Настройка Firefox для работы с VPN

## ❌ Проблема:
Firefox не может открыть HTTPS сайты через VPN с ошибкой `PR_END_OF_FILE_ERROR`

## ✅ Решение:

### СПОСООБ 1: Правильная настройка Firefox

#### 1. Откройте настройки:
```
about:preferences
```

#### 2. Настройка прокси:
1. Прокрутите вниз до **Параметры сети**
2. Нажмите **Настроить...**
3. Выберите **Ручная настройка прокси**
4. Заполните:
   - **SOCKS5 прокси:** `127.0.0.1`
   - **Порт:** `10808`
   - ✅ **DNS через SOCKS v5**
5. **Уберите галочку** с "Также использовать этот прокси для HTTPS"
6. Нажмите **OK**

#### 3. Отключите DNS-over-HTTPS:
1. В настройках найдите **DNS**
2. Отключите **Защищённый DNS (DNS over HTTPS)**
3. Или выберите **Максимальная защита**

#### 4. Отключите WebRTC:
```
about:config
media.peerconnection.enabled = false
```

---

### СПОСОБ 2: Использование HTTP прокси (РЕКОМЕНДУЕТСЯ)

#### Настройка:
1. Настройки → Параметры сети
2. **HTTP прокси:** `127.0.0.1`
3. **Порт:** `10809`
4. **SSL прокси:** `127.0.0.1`
5. **Порт:** `10809`
6. OK

**Преимущества:**
- ✅ Работает HTTPS
- ✅ DNS не протекает
- ✅ Проще настройка

---

### СПОСОБ 3: Расширение Proxy SwitchyOmega

#### Установка:
1. Откройте **Firefox Add-ons**
2. Найдите **Proxy SwitchyOmega**
3. Установите

#### Настройка:
1. Откройте настройки расширения
2. Добавьте новый профиль: **VPN**
3. Тип: **SOCKS5**
4. Сервер: `127.0.0.1`
5. Порт: `10808`
6. ✅ **Use SOCKS5 for DNS**
7. Сохраните

#### Использование:
- Кликните на иконку расширения
- Выберите профиль **VPN**
- Готово!

---

### СПОСОБ 4: Запуск с параметрами

```bash
firefox \
  --network-proxy-type=manual \
  --network-proxy-socks-host=127.0.0.1 \
  --network-proxy-socks-port=10808 \
  --network-proxy-socks-version=5
```

## ⚠️ ВАЖНО ПОНИМАТЬ

### Reality протокол имеет ограничения:

**Работает:**
- ✅ HTTP сайты через SOCKS5
- ✅ Некоторые HTTPS сайты
- ✅ DNS через SOCKS5h

**Может не работать:**
- ⚠️ Некоторые HTTPS сайты (банки, госуслуги)
- ⚠️ Сайты с strict TLS проверкой

### Решение для HTTPS:

**Используйте HTTP прокси вместо SOCKS5:**
- HTTP: `127.0.0.1:10809`
- Работает стабильнее для HTTPS

## 🆘 ЕСЛИ ВСЁ ЕЩЁ НЕ РАБОТАЕТ

### 1. Очистите кэш Firefox:
```
about:preferences#privacy
→ Куки и данные сайтов → Удалить данные
```

### 2. Сбросьте настройки сети:
```
about:config
network.proxy.* → сбросить
```

### 3. Перезапустите VPN:
```bash
sudo systemctl restart vless-vpn-ultimate
```

### 4. Проверьте VPN:
```bash
# HTTP через VPN
curl -x http://127.0.0.1:10809 http://example.com

# HTTPS через VPN
curl -x http://127.0.0.1:10809 https://example.com
```
