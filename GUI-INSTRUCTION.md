# 🖥️ VLESS VPN Client GUI

## Установка графического интерфейса

### Вариант 1: С sudo (рекомендуется)

```bash
# 1. Установите PyQt5
sudo apt-get update
sudo apt-get install -y python3-pyqt5

# 2. Запустите установщик GUI
cd /home/kostik/vpn-client-aggregator
sudo ./install-gui.sh

# 3. Запустите из меню приложений или командой
vpn-gui
```

### Вариант 2: Без sudo (если PyQt5 уже установлен)

```bash
cd /home/kostik/vpn-client-aggregator
./install-gui-no-sudo.sh
```

### Вариант 3: Ручная установка

```bash
# 1. Установите PyQt5
sudo apt-get install python3-pyqt5

# 2. Запустите GUI напрямую
python3 /home/kostik/vpn-client-aggregator/vpn_gui.py
```

## 🖼️ Возможности GUI

- ✅ **Подключение в 1 клик** — кнопка Подключить/Отключить
- ✅ **Выбор режима** — Split (РФ напрямую) или Full (весь трафик через VPN)
- ✅ **Мониторинг статуса** — автоматическое обновление состояния
- ✅ **Просмотр логов** — в реальном времени
- ✅ **Системный трей** — работа в фоне
- ✅ **Автозапуск** — опция в настройках
- ✅ **Статистика** — количество серверов, онлайн/оффлайн

## ⚙️ Настройки прокси в браузере

Для работы через GUI настройте прокси в браузере:

### Firefox/Chrome с Proxy SwitchyOmega

1. Установите расширение **Proxy SwitchyOmega**
2. Создайте новый профиль:
   - Protocol: **SOCKS5**
   - Server: **127.0.0.1**
   - Port: **10808**
3. Включите профиль после подключения VPN

### Настройки системы

```bash
# Временная настройка
export all_proxy=socks5://127.0.0.1:10808
export https_proxy=http://127.0.0.1:10809

# Проверка
curl https://www.facebook.com -I
```

## 🔧 Решение проблем

### PyQt5 не устанавливается

```bash
# Попробуйте альтернативный метод
sudo apt-get install python3-pip
pip3 install PyQt5
```

### GUI не запускается

```bash
# Проверьте ошибку
python3 /home/kostik/vpn-client-aggregator/vpn_gui.py

# Убедитесь что VPN клиент установлен
ls -la ~/vpn-client/logs/
```

### Иконка в трее не отображается

Установите библиотеку для трея:
```bash
sudo apt-get install libxcb-xinerama0
```
