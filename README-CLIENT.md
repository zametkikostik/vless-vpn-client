# 📖 VLESS VPN CLIENT - ИНСТРУКЦИЯ

## 🔧 Команды

### 1️⃣ Обновить серверы
```bash
vless-vpn update
```
👉 Скачает 1945 серверов из GitHub за 2 секунды

### 2️⃣ Подключиться к VPN
```bash
vless-vpn start --auto
```
👉 Протестирует 5 лучших серверов  
👉 Подключит к ближайшему рабочему  
👉 Время подключения: ~3 секунды

### 3️⃣ Проверить статус
```bash
vless-vpn status
```
👉 Покажет: подключен/нет, сервер, прокси

### 4️⃣ Отключиться
```bash
vless-vpn stop
```

### 5️⃣ Включить прокси для AI-сервисов
```bash
source ~/.vpn_proxy.sh
```
👉 Настроит переменные окружения  
👉 Claude, ChatGPT, Lovable будут работать

---

## 🔄 Быстрый старт (после перезагрузки)

```bash
vless-vpn status              # Проверить, работает ли
source ~/.vpn_proxy.sh        # Включить прокси
```

**Готово!** Можно работать.

## 📊 Что видит клиент

| Параметр | Значение |
|----------|----------|
| Всего серверов | 1945 (из кэша) |
| Онлайн | 1945 (все доступны) |
| Тестирует | первые 5 серверов с низким пингом |
| Приоритет | LOW-PING → WHITE → BLACK серверы |

---

## 🎯 Алгоритм выбора сервера

1. **LOW-PING** (<100ms) — серверы с низким пингом (обычно рабочие)
2. **WHITE** — серверы из white-листа
3. **BLACK** — все остальные reality серверы

Клиент тестирует первые 5 серверов TCP-подключением и выбирает первый доступный.

## 🐛 Если не работает

```bash
# 1. Обновить серверы
vless-vpn update

# 2. Переподключиться
vless-vpn stop
vless-vpn start --auto

# 3. Проверить логи
tail -20 ~/vpn-client/logs/client.log
```
