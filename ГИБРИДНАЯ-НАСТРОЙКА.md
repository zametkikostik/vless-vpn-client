## ⚡ БЫСТРАЯ НАСТРОЙКА FIREFOX

### Шаг 1: Откройте настройки
```
about:preferences
```

### Шаг 2: Параметры сети
1. Прокрутите вниз до **Параметры сети**
2. Нажмите **Настроить...**

### Шаг 3: Настройка прокси
Выберите **Ручная настройка прокси** и заполните:

| Поле | Значение |
|------|---------|
| **HTTP прокси** | `127.0.0.1` |
| **Порт** | `10808` |
| **SSL прокси** | *(оставьте пустым)* |
| **Порт** | *(оставьте пустым)* |
| **SOCKS5 прокси** | *(оставьте пустым)* |
| **Порт** | *(оставьте пустым)* |

### Шаг 4: Исключения
В поле **"Не использовать прокси для"** добавьте:
```
*.youtube.com, *.google.com, *.facebook.com, *.instagram.com, *.twitter.com, api.ipify.org, dnsleaktest.com, whoer.net
```

### Шаг 5: OK

## 🧪 НАСТРОЙКА ДЛЯ РАЗНЫХ САЙТОВ

### Способ 1: Автоматический прокси (PAC)

**Создайте файл** `~/proxy.pac`:
```javascript
function FindProxyForURL(url, host) {
  // HTTPS сайты напрямую
  if (shExpMatch(host, "*.youtube.com")) return "DIRECT";
  if (shExpMatch(host, "*.google.com")) return "DIRECT";
  if (shExpMatch(host, "*.facebook.com")) return "DIRECT";
  if (shExpMatch(host, "*.instagram.com")) return "DIRECT";
  if (shExpMatch(host, "*.twitter.com")) return "DIRECT";
  if (shExpMatch(host, "api.ipify.org")) return "DIRECT";
  if (shExpMatch(host, "dnsleaktest.com")) return "DIRECT";
  
  // Остальные через VPN
  return "SOCKS5 127.0.0.1:10808";
}
```

**В Firefox:**
1. Настройки → Параметры сети
2. **Автоматический прокси**
3. Укажите: `file:///home/kostik/proxy.pac`

---

### Способ 2: Расширение Proxy SwitchyOmega

**Установка:**
1. Firefox Add-ons → Proxy SwitchyOmega
2. Установить

**Настройка:**
1. Создайте профиль **VPN**
   - Тип: SOCKS5
   - Сервер: 127.0.0.1
   - Порт: 10808

2. Создайте профиль **Direct**
   - Тип: Direct connection

3. Настройте **Auto Switch**:
   - HTTPS сайты → Direct
   - Остальные → VPN

**Использование:**
- Кликните на иконку расширения
- Выберите **Auto Switch**
