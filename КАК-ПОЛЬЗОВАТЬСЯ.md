## 🔧 НАСТРОЙКА БРАУЗЕРА

### Firefox:
1. Откройте **Настройки**
2. Прокрутите вниз до **Параметры сети**
3. Нажмите **Настроить...**
4. Выберите **Ручная настройка прокси**
5. Заполните:
   - **SOCKS5 прокси:** `127.0.0.1`
   - **Порт:** `10808`
   - ✅ **DNS через SOCKS v5**
6. Нажмите **OK**

### Chrome/Chromium:
**Запуск с прокси:**
```bash
google-chrome --proxy-server="socks5://127.0.0.1:10808"
```

Или установите расширение:
- **Proxy SwitchyOmega**
- **FoxyProxy**

### Opera:
1. Настройки → Внизу **Показать дополнительные**
2. Система → **Открыть настройки прокси-сервера**
3. Как в Firefox выше

## 📱 НАСТРОЙКА НА ТЕЛЕФОНЕ (Android)

### Через приложение:
1. Установите **v2rayNG** или **Hiddify**
2. Импортируйте конфиг из файла `servers.json`
3. Подключитесь

### Через HTTP прокси:
1. Настройки → Wi-Fi
2. Долгое нажатие на вашу сеть
3. Изменить сеть → Дополнительно
4. Прокси-сервер: **Вручную**
5. Имя хоста: IP вашего компьютера
6. Порт: `10809`

## 🌐 ПРОВЕРКА РАБОТЫ

### Тест HTTP:
```bash
curl -x http://127.0.0.1:10809 http://example.com
```

### Тест SOCKS5:
```bash

```

### Узнать IP:
```bash
curl --socks5 127.0.0.1:10808 https://api.ipify.org?format=text
```
